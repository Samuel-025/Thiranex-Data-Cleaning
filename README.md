Thiranex Task 1: Data Cleaning & Visualization

This repository contains the first project of the Thiranex Data Science curriculum. The primary goal was to process a raw, "messy" dataset to make it suitable for analysis and to visualize initial insights.

🎯 Project Goals

Handle missing values using statistical imputation.

Identify and remove duplicate records.

Detect and manage outliers in numerical data.

Visualize demographic insights using Python.

🛠️ Tech Stack

Pandas: For data manipulation and cleaning.

Seaborn & Matplotlib: For creating statistical visualizations.

📊 Key Findings

Cleaning: Successfully imputed missing Age values using the median and Embarked values using the mode.

Outliers: Capped the Fare column to ensure visualizations were not skewed by extreme values.

Insight: Data analysis revealed a significant survival advantage for female passengers and those in higher classes.
